#-----------------------------------------
#	操作方法
#-----------------------------------------
・移動(ダッシュ)
PAD:左スティックを半分以上倒す
キーボード・マウス:WASD

・移動(歩き)
PAD:左スティックを少し倒す
キーボード・マウス:WASD+SHFIT

・クイックターン
PAD:ダッシュ中にRB
キーボード・マウス:ダッシュ中に右クリック

・ジャンプ
PAD:A
キーボード・マウス:SPACE

#-----------------------------------------
#	現状の実装内容
#-----------------------------------------
ダッシュとクイックターン
ブーストジャンプ
壁ジャンプ
